# ECG Monitoring System

## 📌 Project Overview

The ECG Monitoring System is a biomedical engineering project developed to monitor and analyze the electrical activity of the human heart in real time. The system uses an AD8232 ECG Sensor Module along with an Arduino Uno/ESP32 microcontroller to acquire ECG signals, process them, calculate heart rate, and display ECG waveforms.

This project provides a portable, low-cost, and efficient solution for continuous cardiac monitoring. It can be used for educational purposes, healthcare research, and basic patient monitoring applications.

---

## 🎯 Objectives

- Acquire ECG signals from the human body.
- Monitor heart activity in real time.
- Calculate heart rate in Beats Per Minute (BPM).
- Display ECG waveform and heart rate.
- Provide a portable and economical healthcare solution.
- Enable future integration with IoT and cloud technologies.

---

## ❗ Problem Statement

Cardiovascular diseases are among the leading causes of death worldwide. Traditional ECG monitoring systems are often expensive, bulky, and limited to hospitals and diagnostic centers.

There is a need for a portable and affordable ECG monitoring system that can continuously monitor heart activity, detect abnormalities, and support remote healthcare monitoring.

The proposed ECG Monitoring System addresses these challenges by providing real-time cardiac monitoring using embedded system technology.

## ⚙ Working Principle

### Step 1: Signal Acquisition
ECG electrodes are attached to the human body to detect electrical signals generated by heart activity.

### Step 2: Signal Conditioning
The AD8232 sensor module amplifies weak ECG signals and removes unwanted noise using built-in filters.

### Step 3: Signal Processing
The microcontroller receives the conditioned signal and converts it into digital data using the ADC.

### Step 4: Heart Rate Calculation
The system detects ECG peaks and calculates heart rate in BPM.

### Step 5: Display
The ECG waveform and BPM value are displayed on an LCD/OLED screen.

### Step 6: Wireless Communication (Optional)
The ECG data can be transmitted to:
- Mobile Application
- Computer Dashboard
- Cloud Platform

## 📋 Algorithm

1. Start the system.
2. Initialize sensor and display modules.
3. Read ECG signal from AD8232.
4. Convert analog signal into digital values.
5. Filter noise.
6. Detect ECG peaks.
7. Calculate BPM.
8. Display ECG waveform and BPM.
9. Send data through Bluetooth/Wi-Fi (optional).
10. Repeat continuously.
